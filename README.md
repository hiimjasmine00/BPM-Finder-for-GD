**BPM Detector**

A little mod that adds a Find BPM button in the BPM trigger.
Can help creators with synchronization.

Long songs won't work unfortunately, aim for a song less than 5 minutes.


Dependencies / Credits:
 - minibpm: https://github.com/breakfastquay/minibpm
 - minimp3: https://github.com/lieff/minimp3
